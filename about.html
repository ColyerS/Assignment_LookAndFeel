ABOUT PROJECT

I. (3 point) Responsiveness – compatibility with small (mobile), medium (tablet) and large (desktop) screen sizes
Whether on a mobile phone/tablet/large screen, the pages expand the content to fit the size of the screen. The user can easily navigate to other pages since the navigation bar shifts to be either a list format or a row depending on the screen size. Similiarly all the images and text shirnk and contract when the view is altered.

II. (3 points) Usability – ease of use and navigation of your interface end users
The website has a top navigation for each page that changes color when hovering to facilitate selection of the page. The navigation is consistent throughout the site. If the end user it at the bottom of the page, there are also links to each page to navigate through the footer of the page. This allows the end user to easily go through each of the sites and also go back to previous sites.

III. (6 points) Content – relevance of the content on the site
The website is a simple tool to view the most traveled to countries in the world and then see those top three countries facts/figures. The images chosen are of famous landmarks of the country in question to entice the viewer to go and explore the country themselves. The facts and figures are meant as a stepping stone in the research behind traveling to those countries.

IV. (4.5 points) Design – use of at least 3 specific design guidelines and ideas for web sites

  • The website has a simple but inviting color scheme, not overtly distracting but warm and slightly interesting.
  • The website has clean lines, there are minimal distractions, the same layout is used from page to page to create consistency in the experience and simplicity.
  • The website uses white space, allowing the user to focus on the content at hand.

V. (1.5 points) Please add a comment to your submission, reflecting on your learnings, total time required and any recommendations for future assignments
The assignment took about 6 hours as a group of 3 to complete. We have learned how to write css and html for a simple website as well as imbedding pictures and charts based on data. We would recommend to have more overview of the basics of html and css in class prior to having the assignment to assist with familiarity
